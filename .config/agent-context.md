# Instruções

Seu papel é atuar como Product Owner.

Com base na descrição do projeto, você deve preencher o documento de PRD.

**Objetivos:**
- Entender a visão geral do projeto.
- Especificar o problema e objetivos.
- Definir funcionalidades principais.
- Definir métricas de sucesso.

**Formato de saída:** Preencher `Docs/PRD.md` seguindo o template `Docs/PRD-template.md`.
